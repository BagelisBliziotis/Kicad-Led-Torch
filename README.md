# LED Torch / My First KiCad PCB Design


*Features
- Single-layer PCB 
- Compact and efficient design
- Fully annotated schematic with clear net labels
- 3D render included
- Complete manufacturing files (Gerbers + Drill) ready for order.

*How to open the project
1. download this repository
2. Open the file `led-torch.kicad_pro` with KiCad 10.0

*Manufacturing files
All Gerber and drill files are inside the Gerbers for led torch folder.  
You can upload the entire folder directly to any PCB manufacturer (JLCPCB, PCBWay, etc.).

*Why this project?
- First step into real PCB design after finishing the Arduino phase
- Practising KiCad workflow (schematic → PCB → Gerbers)
- Learning proper documentation and version control 
- Building towards a full STM32-based telemetry system this summer
*What i learned:
- Complete KiCad workflow which was actually the hardest part in this project.From schematic capture → footprint assignment → PCB layout → DRC/ERC checks → Gerber & drill file generation    
- Importance of clean net labels ground  and component placement
- Difference between breadboard/Arduino prototyping and a real custom PCB
